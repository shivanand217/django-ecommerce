
* Create a virtual environment to start a new project in isolation
  Steps to create on this mac with python3.6

- cd /Library/Frameworks/Python.framework/Versions/3.6/lib/python3.6/site-packages 
- Run the script like this: python virtualenv.py --distribute /the/path/at/which/you/want/the/new/venv/at theNameOfTheNewVirtualEnv.

* Start a new project   
  django-admin startproject <project-name> .
  . at the end to not create subdirectory project structure

* For Creating Super User
  python manage.py createsuperuser <super-user-name>

* Starting the Django Server
  python manage.py runserver

  To start on a specific port number
  python manage.py runserver <port-number>

* Starting a new App
  python manage.py startapp <app-name>

* Every Single Time you make some models changes run commands
  python manage.py makemigrations
  python manage.py migrate

* After making migrations of the models for an app register those models in app's admin.py file 
  so that our admin can access those models from the admin dashboard.
  - import your models in the admin file
  - do admin.site.register(<model-name>)
